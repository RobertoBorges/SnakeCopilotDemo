# How to Run the Snake Game

## Quick Start Guide

### 1. Install Dependencies
```bash
npm install
```

### 2. Build the Project
```bash
npm run build
```

### 3. Start the Game
```bash
npm start
```

### 4. Play the Game
- Open your browser to `http://localhost:3000`
- Click "Start Game" 
- Use arrow keys to control the snake
- Eat red food to grow and score points
- Avoid walls and your own body!
